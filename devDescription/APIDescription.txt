UserAPI

# Login


Post /login

Request
{
    "id" : "UserId",
    "pw" : "1234"
}

Response 성공시 유저 정보 반환
{
    "login" : true | false
    "user" : {
                    "id" : "UserID",
                    "pw" : "1234",
                    "UserLevel" : "Admin"
                    ...
             }

    // 상태코드 200

}

# Objects

- 오브젝트 정보 불러오기
Get /objects/:objectName

Response 성공시 객체List 반환
{
    "objectName" : ":objectName",
    objects :
    [
        {...},
        {...}
    ]
}

- 오브젝트 정보 C U D
Post /objects/:objectName

Request
{
    "objectName" : ":objectName",
    "action" : "CREATE | UPDATE | DELETE"
    "objects" :
    [
        {...},
        {...}
    ]
}

Response 성공시 전체 객체List 반환
{
    "objectName" : ":objectName",
    objects :
    [
        {...},
        {...}
    ]
}
